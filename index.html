Kivert Helm Chart Repository

This is the official Helm chart repository for Kivert, a Kubernetes pod-restart alerting controller.

How to Install

Add this repository to your Helm configuration:

helm repo add kivert https://colt005.github.io/kivert
helm repo update

Install the Kivert controller:

helm install kivert kivert/kivert \
  --namespace kivert-system \
  --create-namespace

For more detailed configuration options, view the Kivert GitHub Repository.
